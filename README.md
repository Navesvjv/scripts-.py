## Black Hat - Python (socket)

#### For now the server only works on Linux.

<br>

## Usage:

Server
```
python socket.py -l -a "IP:PORT"
```

Client
```
python socket.py -i -a "IP:PORT"
```

<br>

## parameters:

- ``-l or --listen`` <br>
It becomes a server.

- ``-i or --iterable`` <br>
It becomes a client.

- ``-a or --address`` <br>
Target address.
